AVL Tree Theory

Introduction

An AVL Tree is a self-balancing Binary Search Tree. In an AVL Tree, the height difference between left and right subtrees of any node is at most 1. This ensures operations like search, insertion, and deletion run in O(log n) time.

Need for AVL Tree

  • Normal BST can become skewed depending on insertion order
  • Skewed BST leads to inefficient search (O(n))
  • AVL Trees maintain balance automatically to prevent skewed structures

Balance Factor

Balance Factor = Height(left subtree) − Height(right subtree)

Balance FactorStatus
-1, 0, +1Balanced
<-1 or >1Unbalanced

Rotations in AVL Trees

  • LL Rotation – Left-Left imbalance
  • RR Rotation – Right-Right imbalance
  • LR Rotation – Left-Right imbalance
  • RL Rotation – Right-Left imbalance
