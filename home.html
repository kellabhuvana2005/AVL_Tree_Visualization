Welcome to AVL Tree Visualization Project

This project provides an interactive visualization of AVL Trees, a self-balancing Binary Search Tree that maintains height balance automatically. It allows students to understand insertion, deletion, rotations, and tree balance in real-time.

Objectives

  • Visualize AVL Tree operations dynamically
  • Observe rotations (LL, RR, LR, RL)
  • See node heights and balance factors
  • Interactive learning experience for data structures
