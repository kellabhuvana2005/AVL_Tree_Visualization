AVL Tree Algorithms

Insertion Algorithm

  1. Insert node like in a Binary Search Tree
  2. Update heights of ancestor nodes
  3. Check balance factor
  4. If unbalanced, perform appropriate rotation (LL, RR, LR, RL)

Deletion Algorithm

  1. Delete node like in BST
  2. Update heights
  3. Check balance factor
  4. If unbalanced, perform rotation (LL, RR, LR, RL)

Time Complexity

OperationTime Complexity
SearchO(log n)
InsertO(log n)
DeleteO(log n)
